# User Personas

## 1) Aisha – Gen Z Online Shopper (22)
- Goals: Quickly preview trendy items, share looks with friends.
- Behaviors: Mobile-first, short sessions, expects instant feedback and smooth UI.
- Pain Points: Slow uploads, confusing controls, non-intuitive overlays.

## 2) Rahul – Fashion Blogger (28)
- Goals: Create content quickly, test combinations, export clean images.
- Behaviors: Desktop usage, values shortcuts, wants download and fine-tuning (scale/rotate).
- Pain Points: Low-quality exports, lack of favorites, poor item organization.

## 3) Neha – Casual Shopper (33)
- Goals: Find a good fit visually before purchase, save a few items.
- Behaviors: Mix of mobile/desktop, appreciates guidance and defaults.
- Pain Points: Overwhelming options, no onboarding, non-responsive layouts.
