# Wireframes (Low-Fi Outline)

## Home
- Hero: Title + short description
- Primary CTA: Start Try-On
- Secondary CTA: Browse Catalog

## Catalog
- Grid of items (image, name, category)
- Actions: Save (favorites), Try (navigates to Try On and sets selected item)

## Try On
- Left Panel: Upload, Webcam toggle, Item chips, Transform controls (scale, rotate, reset), Save Item, Download Look
- Right Stage: 3:4 canvas showing uploaded image or webcam feed; overlay draggable with mouse; centered by default

## Favorites
- Grid of saved items with ability to remove

## Notes
- Mobile-first: Stack panels vertically; stage width 100%; controls fold into accordions if needed (future)
- Accessibility: Clear labels, sufficient color contrast, keyboard focus styles (future)
